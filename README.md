# ai-selenium-automation-claude-sdet
Selenium + CI/CD + Claude AI usage
AI-Assisted Selenium Automation Framework (Claude + SDET)

This framework demonstrates:
- Selenium WebDriver automation using Page Object Model (POM)
- AI-assisted test design using Claude Opus
- CI/CD integration (Azure DevOps / GitHub Actions)
- API + UI automation coverage
- Log analysis using GenAI

AI capabilities:
- Test case generation from requirements
- Edge-case expansion
- Automation script suggestions
- Failure log analysis

Tech Stack:
- C# / .NET
- Selenium WebDriver
- NUnit / MSTest
- AI (Claude Opus)
- CI/CD pipelines
